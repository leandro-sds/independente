# Independent Theme

**Tema WordPress moderno, responsivo e acessível — desenvolvido por e para pessoas com deficiência visual.**

O Independent Theme nasceu da experiência real de um desenvolvedor cego construindo sites para outros cegos. Cada detalhe foi pensado para funcionar bem com leitores de tela, manter contraste WCAG AA em todos os estilos e oferecer uma experiência de navegação digna para todos.

---

## 🎨 13 Estilos Visuais

Selecionáveis em **Aparência → Personalizar → Estilo Visual**. A troca é instantânea — nenhum conteúdo, menu ou widget é perdido. Só mudam as cores, fontes, animações e identidade visual.

Cada estilo é carregado sob demanda — o visitante baixa apenas o CSS do estilo ativo, sem desperdício.

## ♿ Acessibilidade

O Independent Theme foi construído com acessibilidade como prioridade, não como recurso adicional:

- Todos os pares de cor/fundo validados com **contraste WCAG AA** (mínimo 4.5:1)
- Menu mobile **oculto do leitor de telas** quando recolhido (`visibility: hidden`)
- Imagens com `alt` descritivos
- Botões e links com estados de foco visíveis (`focus-visible`)
- Estrutura semântica HTML5 com landmarks ARIA
- Respeita `prefers-reduced-motion` — animações desativadas para quem preferir
- Testado com **NVDA** em Windows

## 🚀 Recursos

- **13 estilos visuais** com identidade própria, carregados sob demanda
- **Imagem destacada** normalizada — altura máxima 350px com `object-fit: cover`
- **Imagens no corpo do post** — largura total, altura máxima 500px, sem sobreposição de texto
- **Menu mobile** com transição suave e acessível para leitores de tela
- **Categorias e subcategorias** — exibe "Seções" quando há subcategorias; posts quando não há
- **Ordenação de categorias** compatível com o plugin "Category Order and Taxonomy Terms Order"
- **Crédito do desenvolvedor** personalizável no rodapé (nome, URL, opção de ocultar)
- **Logo personalizada** com controle de largura, altura e escala via Personalizador
- **Busca no cabeçalho** estilizada por estilo visual
- **Integração com Player Radio** — botão de play, barras do visualizador e ponto AO VIVO assumem automaticamente a cor de acento do estilo ativo
- **Ícones de redes sociais** — WhatsApp, Facebook, Instagram, YouTube
- **Posts relacionados** por categoria no rodapé do post
- **Botão ← Voltar** estilizado após o conteúdo
- **Paginação** estilizada por estilo visual
- **Comentários** com template próprio e formulário estilizado
- **Página 404** personalizada
- **Conteúdo principal** com altura mínima de 60vh — sem vazios visuais em páginas curtas
- **Atualizações automáticas** via GitHub Releases
- Layout responsivo — desktop, tablet e mobile

## 📦 Instalação

1. Faça upload da pasta `independent-theme` em `wp-content/themes/`
2. Ative o tema em **Aparência → Temas**
3. Acesse **Aparência → Personalizar** para configurar o estilo e identidade

## 🔄 Atualizações via GitHub

O tema se atualiza automaticamente pelo painel do WordPress quando uma nova versão for publicada no GitHub em **Releases**. O arquivo ZIP do release deve se chamar exatamente `independent-theme-X.X.zip` (ex: `independent-theme-5.4.zip`).

# Independente Theme

**Tema WordPress moderno, responsivo e acessível — desenvolvido por e para pessoas com deficiência visual.**

O Independente nasceu da experiência real de um desenvolvedor cego construindo sites para outros cegos. Cada detalhe foi pensado para funcionar bem com leitores de tela, manter contraste WCAG AA em todos os estilos e oferecer uma experiência de navegação digna para todos os usuários — com ou sem deficiência.

> _"Criado por uma pessoa cega para pessoas cegas desenvolverem sites para quem enxerga."_

---

## 🎨 7 Estilos Visuais

Selecionáveis em **Aparência → Personalizar → Estilo Visual**. A troca é instantânea — nenhum conteúdo, menu ou widget é perdido. Só mudam as cores, fontes, animações e identidade visual.

Cada estilo é carregado sob demanda — o visitante baixa apenas o CSS do estilo ativo, sem desperdício.

## ♿ Acessibilidade

O Independente foi construído com acessibilidade como prioridade, não como recurso adicional:

- Todos os pares de cor/fundo validados com **contraste WCAG 2.1 AA** (mínimo 4.5:1) em todos os 7 estilos
- Skip-link para saltar direto ao conteúdo principal
- Menu mobile **oculto do leitor de telas** quando recolhido
- Botões e links com estados de foco visíveis (`focus-visible`) — anel de foco personalizado por estilo
- Estrutura semântica HTML5 com landmarks ARIA
- Respeita `prefers-reduced-motion` — animações desativadas para quem preferir
- Suporte a idiomas **RTL** (árabe, hebraico)
- Testado com **NVDA** em Windows

## 🚀 Recursos

- **7 estilos visuais** com identidade própria, animações exclusivas e carregamento sob demanda
- **Fontes self-hosted** — Inter, Montserrat, Poppins, Playfair Display, Plus Jakarta Sans, Raleway e Oswald incluídas no tema, sem chamadas ao Google Fonts (privacidade do visitante preservada)
- **Imagem destacada** normalizada — altura máxima com `object-fit: cover`
- **Menu principal** com suporte a navegação completa por teclado
- **Menu mobile** com transição suave e acessível para leitores de tela
- **Busca no cabeçalho** estilizada por estilo visual
- **Integração com Player Radio** — botão de play e barras do visualizador assumem a cor de acento do estilo ativo
- **Ícones de redes sociais** — WhatsApp, Facebook, Instagram, YouTube
- **Posts relacionados** por categoria no rodapé do post
- **Paginação** estilizada por estilo visual
- **Comentários** com template próprio e formulário estilizado
- **Página 404** personalizada
- **Logo personalizada** com controle de largura, altura e escala via Personalizador
- **Crédito do desenvolvedor** personalizável no rodapé
- Layout responsivo — desktop, tablet e mobile

## 📦 Instalação

1. Faça o download do arquivo `.zip` na seção [Releases](https://github.com/leandro-sds/independente/releases)
2. No painel WordPress, vá em **Aparência → Temas → Adicionar novo → Enviar tema**
3. Ative o tema
4. Acesse **Aparência → Personalizar** para configurar o estilo visual e a identidade do site

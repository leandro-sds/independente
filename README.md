# Independent Theme

**Tema WordPress moderno, responsivo e acessível — desenvolvido por e para pessoas com deficiência visual.**

O Independent Theme nasceu da experiência real de um desenvolvedor cego construindo sites para outros cegos. Cada detalhe foi pensado para funcionar bem com leitores de tela, manter contraste WCAG AA em todos os estilos e oferecer uma experiência de navegação digna para todos.

---

## 🎨 9 Estilos Visuais

Selecionáveis em **Aparência → Personalizar → Estilo Visual**. A troca é instantânea — nenhum conteúdo, menu ou widget é perdido. Só mudam as cores, animações e identidade visual.

| Estilo | Ideal para |
|---|---|
| ⬜ **Padrão** | Qualquer tipo de site — neutro e elegante |
| 🎧 **Neon Pop** | Rádio jovem, podcast, DJ, games, streamers |
| 📻 **Vintage Café** | Rádio retrô, jazz, blues, cafeterias, saudosismo |
| ⚽ **Campo e Paixão** | Futebol, esportes, torcidas, academias, jornalismo esportivo |
| ✝️ **Céu e Fé** | Rádio gospel, igrejas, ministérios, sites espirituais |
| ✍️ **Tinta & Papel** | Escritores, blogs literários, poesia, revistas, jornalismo |
| 🏛️ **Marinelli** | Portais de notícias, sites institucionais, rádios, prefeituras |
| ⚡ **Moderno** | Startups, agências digitais, e-sports, artistas, criadores |
| 🔴 **Colorado** | Clubes de futebol, torcidas, academias com identidade vermelho e branco |

## ♿ Acessibilidade

O Independent Theme foi construído com acessibilidade como prioridade, não como recurso adicional:

- Todos os pares de cor/fundo validados com **contraste WCAG AA** (mínimo 4.5:1)
- Menu mobile **oculto do leitor de telas** quando recolhido (`visibility: hidden`)
- Imagens destacadas com `aria-label` e `alt` descritivos
- Botões e links com estados de foco visíveis
- Estrutura semântica HTML5 com landmarks ARIA
- Respeita `prefers-reduced-motion` — animações desativadas para quem preferir
- Testado com **NVDA** em Windows

## 🚀 Recursos

- **9 estilos visuais** prontos para uso com identidade própria
- **Imagens destacadas** na listagem (pequena, à esquerda) e no post (grande, no topo)
- **Menu mobile** com transição suave e acessível para leitores de tela
- **Categorias e subcategorias** — exibe "Seções" quando há subcategorias; posts quando não há
- **Ordenação de categorias** compatível com o plugin "Category Order and Taxonomy Terms Order"
- **Crédito do desenvolvedor** personalizável no rodapé (nome, URL, opção de ocultar)
- **Logo personalizada** com controle de largura, altura e escala
- **Busca no cabeçalho** com campo e botão estilizados por estilo
- **Ícones de redes sociais** — WhatsApp, Facebook, Instagram, YouTube
- **Posts relacionados** por categoria no rodapé do post
- **Botão ← Voltar** estilizado após o conteúdo
- **Paginação** estilizada por estilo visual
- **Comentários** com template próprio e avatar
- **Widget de comentários recentes** corrigido para não transbordar
- **Página 404** personalizada
- **Atualizações automáticas** via GitHub Releases
- Layout responsivo — desktop, tablet e mobile

## 📦 Instalação

1. Faça upload da pasta `independent-theme` em `wp-content/themes/`
2. Ative o tema em **Aparência → Temas**
3. Acesse **Aparência → Personalizar** para configurar

## 🔄 Atualizações via GitHub

O tema se atualiza automaticamente pelo painel do WordPress quando uma nova versão for publicada no GitHub em **Releases**. O arquivo ZIP do release deve se chamar exatamente `independent-theme-X.X.X.zip` (com pontos, não underscores).

# Estilos Visuais — Independent Theme v4.3.0

Cada estilo foi projetado com identidade própria, animações exclusivas e personalidade visual
única — mantendo o mesmo layout, acessibilidade e funcionalidades em todos eles.

**Como trocar:** Aparência → Personalizar → Estilo Visual → Publicar

A mudança é instantânea e reversível. Nenhum conteúdo é perdido.

Todas as animações respeitam `prefers-reduced-motion` — quem prefere sem animações, navega sem elas.

Todos os pares de cor/fundo foram validados com contraste WCAG AA (mínimo 4.5:1).

## ⬜ Padrão

**Ideal para:** sites institucionais, projetos sem identidade visual definida, qualquer tipo de site.

- Menu hover com fundo dourado transparente subindo suavemente
- Títulos com entrada suave — fadeIn universal

## Recursos comuns a todos os estilos

- Layout responsivo — desktop, tablet e mobile
- Imagem destacada na listagem (pequena, à esquerda) e no post (grande, no topo)
- Menu mobile com transição suave e acessível para leitores de tela
- Categorias exibem "Seções" quando têm subcategorias; posts quando não têm
- Ordenação de categorias compatível com plugin de ordenação
- Crédito do desenvolvedor personalizável no rodapé
- Botão ← Voltar estilizado após o conteúdo
- Paginação estilizada
- Posts relacionados por categoria
- Widget de comentários recentes sem transbordamento
- Contraste WCAG AA validado em todos os pares cor/fundo
- Animações respeitam `prefers-reduced-motion`
